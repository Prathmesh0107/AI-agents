# AI Portfolio Manager 🚀

## Overview
An AI-powered agent that fetches real-time stock prices, updates portfolio data in Google Sheets, and provides intelligent responses based on user queries.

## Features
- 📈 Live stock price tracking (Marketstack API)
- 📊 Automatic Google Sheets updates
- 💰 Profit/Loss calculation
- 🤖 AI chat-based interaction
- 🔄 Multi-stock support

## Tech Stack
- OpenRouter (LLM)
- Marketstack API
- Google Sheets API
- Workflow Automation (n8n / similar)

## How It Works
1. User sends a message (e.g., "Get AAPL price")
2. AI agent processes intent
3. Fetches stock data via API
4. Updates Google Sheet
5. Returns response

## Setup Instructions
- Import JSON workflow
- Add API keys (Marketstack, Google)
- Run workflow
